# Functions, Nested Loops

This project introduces:
- How to use nested loops
- How to use function prototypes
- How to use header files
- How to write and use your own functions

## Files

- **0-putchar.c**  
  Prints `_putchar` followed by a new line.

- **_putchar.c**  
  Provided file used to print characters to the standard output.

- **main.h**  
  Header file containing function prototypes for the project.

## Compilation

Compile using:

```
gcc -Wall -pedantic -Werror -Wextra -std=gnu89 _putchar.c 0-putchar.c -o 0-putchar
```

## Usage

```
./0-putchar
```

Output:

```
_putchar
```

C - File I/O
Description
This project covers the use of POSIX system calls to handle files in C, specifically focusing on opening, reading, writing, and closing files using file descriptors.

Key Concepts
System Calls: Using open, read, write, and close.

File Descriptors: Standard Input (0), Standard Output (1), and Standard Error (2).

Flags: Using bitwise OR to combine O_RDONLY, O_WRONLY, O_CREAT, and O_APPEND.

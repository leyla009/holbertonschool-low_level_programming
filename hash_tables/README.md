C - Hash tables

Description

This project covers the implementation and use of Hash Tables in C. It includes creating the hash table structure, implementing the DJB2 algorithm for hashing, and handling collisions using the "Singly Linked List" chaining method.

Requirements
Allowed editors: vi, vim, emacs

Compiled on Ubuntu 20.04 LTS using gcc

Compiler flags: -Wall -Werror -Wextra -pedantic -std=gnu89

Code must follow the Betty style guide.

Compilation
To compile the hash table creation task:

Bash
gcc -Wall -pedantic -Werror -Wextra -std=gnu89 0-main.c 0-hash_table_create.c -o a

# Variadic Functions

This directory contains tasks related to variadic functions in C.

## Files

- **0-sum_them_all.c**  
  Implements a function that returns the sum of all its parameters using
  variadic arguments.

## Concepts Covered

- `stdarg.h`
- `va_list`, `va_start`, `va_arg`, `va_end`
- Handling variable numbers of function arguments

# Pointers, Arrays and Strings

This project introduces basic concepts related to pointers, arrays, and strings in C.  
It is part of the low-level programming track at Holberton School.

## Learning Objectives

- Understand what pointers are and how to use them
- Learn how to manipulate variables through pointers
- Understand the relationship between pointers and arrays
- Perform basic operations on strings

## Project Files

- **0-reset_to_98.c**  
  Function that takes a pointer to an `int` and updates the value it points to to `98`.

- **main.h**  
  Header file containing all function prototypes used in this project.
